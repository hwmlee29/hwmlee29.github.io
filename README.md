# Hyewon Lee Homepage

Static personal website for GitHub Pages.

## Edit the site

- `index.html` contains the page content.
- `styles.css` contains the visual design.
- `script.js` contains the small footer year script.
- `assets/` contains the favicon and monogram artwork.

## Publish with GitHub Pages

This site is ready to publish from the repository root. In GitHub, go to:

`Settings` -> `Pages` -> `Build and deployment` -> `Deploy from a branch`

Then choose the `main` branch and `/ (root)` folder.

For a user site at `https://hwmlee29.github.io/`, GitHub expects the repository
name to be `hwmlee29.github.io`. The current repository name,
`hyewonlee.github.io`, can be published as a project site unless you configure a
custom domain.
